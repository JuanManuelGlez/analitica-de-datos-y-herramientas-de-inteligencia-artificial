Logo Elektra¶

La empresa que me tocó analizar es Electra, la cual es conocida por sus colores amarillos y rojos agresivos.

La paleta de color que posee es la siguiente.

# Crear rectángulos de color

   rect = patches.Rectangle((i, 0), 1, 1, linewidth=1, edgecolor='none', facecolor=color)

La paleta de colores del logo de Elektra, que incluye tonos vibrantes de rojo, amarillo y naranja, está diseñada para captar la atención y evocar emociones de energía, entusiasmo y dinamismo. El rojo dominante sugiere pasión y urgencia, ideal para atraer clientes y destacar en el mercado. Los tonos amarillos añaden una sensación de optimismo y positividad, mientras que el naranja combina la energía del rojo y la alegría del amarillo, creando una paleta equilibrada que transmite una imagen vibrante y acogedora. Esta combinación de colores no solo es visualmente atractiva, sino que también refuerza la identidad de Elektra como una marca enérgica y confiable.

input_dropdown = alt.binding_radio(
    # Add the empty selection which shows all when clicked
    options=options + [None],
    labels=labels + ['All'],
    name='Region: '
)
selection = alt.selection_point(
    fields=['Region'],
    bind=input_dropdown,
)

alt.Chart(df).mark_point().encode(
    x='Sales:Q',
    y='Profit:Q',
    # We need to set a constant domain to preserve the colors
    # when only one region is shown at a time
    color=alt.Color('Region:N').scale(range=logo),
).add_params(
    selection
).transform_filter(
    selection
).interactive()

# Crear un menú desplegable para seleccionar el eje X
dropdown = alt.binding_select(
    options=['Category', 'Region', 'Sub-Category'],
    name='X-axis column '
)
xcol_param = alt.param(
    value='Category',
    bind=dropdown
)

# Crear la gráfica de ventas
chart = alt.Chart(df).mark_circle(size=60).encode(
    x=alt.X('x:N').title(''),
    y=alt.Y('Sales:Q', title='Sales'),
    color=alt.Color('Segment:N', title='Segment', scale=alt.Scale(range=cuadrados)),
    tooltip=['Category:N', 'Region:N', 'Sub-Category:N', 'Sales:Q']
).transform_calculate(
    x=f'datum[{xcol_param.name}]'
).add_params(
    xcol_param
).properties(
    title='Sales by Selected X-axis'
).interactive()

# Mostrar la gráfica
chart.show()

Esta grafica esta genial, ya que son tres graficas en una, donde podemos ver las ventas por categoria, sub-categoria o región según nos parezca. Tambien esta coloreada por segmento, usando la paleta de Cuadrados.

Podemos ver que tiene 3 dimensiones cada una de la grafica, por lo cual aporta muchisima información y nos puede ayudar a sacar buenas concluciónes al respecto.

# Paletas de colores (ejemplo)
triada = ["#F23DC7", "#3DF2F0", "#F2D43D", "#9D925C", "#73566C"]
cuadrados = ["#F2493D", "#443DF2", "#F2D43D", "#3DF289", "#9D925C"]
logo = ["#F22233", "#F2E527", "#F2D43D", "#F2A03D", "#D93814"]
monocromatico = ["#BFAE56", "#8C845B", "#F2D43D", "#59574C", "#333126"]

# Gráfica de ventas con menú desplegable
dropdown = alt.binding_select(
    options=['Category', 'Region', 'Sub-Category'],
    name='X-axis column '
)
xcol_param = alt.param(
    value='Category',
    bind=dropdown
)

sales_chart = alt.Chart(df).mark_circle(size=60).encode(
    x=alt.X('x:N').title(''),
    y=alt.Y('Sales:Q', title='Sales'),
    color=alt.Color('Segment:N', title='Segment', scale=alt.Scale(range=cuadrados)),
    tooltip=['Category:N', 'Region:N', 'Sub-Category:N', 'Sales:Q']
).transform_calculate(
    x=f'datum[{xcol_param.name}]'
).add_params(
    xcol_param
).properties(
    title='Sales by Selected X-axis',
    width=500,
    height=300
).interactive()

# Gráfica de punto con filtro de región
options = ['Central', 'East', 'South', 'West']
labels = [option + ' ' for option in options]

input_dropdown = alt.binding_radio(
    options=options + [None],
    labels=labels + ['All'],
    name='Region: '
)
selection = alt.selection_point(
    fields=['Region'],
    bind=input_dropdown,
)

scatter_plot = alt.Chart(df).mark_point().encode(
    x='Sales:Q',
    y='Profit:Q',
    color=alt.Color('Region:N', scale=alt.Scale(range=logo)),
    tooltip=['Region:N', 'Sales:Q', 'Profit:Q']
).add_params(
    selection
).transform_filter(
    selection
).properties(
    title='Sales and Profit by Region',
    width=500,
    height=300
).interactive()

# Pie chart of region
pie_chart = alt.Chart(df).mark_arc(innerRadius=50).encode(
    theta=alt.Theta(field='count', type='quantitative'),
    color=alt.Color("Region:N", scale=alt.Scale(range=monocromatico)),
    tooltip=['Region:N', alt.Tooltip('count()', title='Count')]
).properties(
    title='Region',
    width=300,
    height=300
).transform_aggregate(
    count='count()',
    groupby=['Region']
)

# Bar chart of category and count
bar_chart_category = alt.Chart(df).mark_bar().encode(
    x=alt.X('Category:N', title='Category'),
    y=alt.Y('count()', title='Count'),
    color=alt.Color("Category:N", scale=alt.Scale(range=logo), legend=None),
    tooltip=['Category:N', alt.Tooltip('count()', title='Count')]
).properties(
    title='Category Count',
    height=400,
    width=600
)

# Concatenar todas las gráficas en un solo tablero
final_dashboard = alt.vconcat(
    sales_chart,
    alt.hconcat(
        scatter_plot,
        pie_chart,
        bar_chart_category
    )
).resolve_scale(
    color='independent'
).configure_mark(
    cornerRadius=3
).configure_view(
    strokeWidth=0
).configure_title(
    fontSize=20,
    anchor='start',
    color='gray'
)

# Mostrar la gráfica
final_dashboard.show()

El tablero interactivo combina múltiples visualizaciones para analizar las ventas de una empresa desde diferentes perspectivas. Incluye una gráfica de barras que muestra las ventas por categoría, una gráfica de burbujas con un menú desplegable que permite seleccionar diferentes dimensiones del eje X (Categoría, Región y Sub-Categoría), una gráfica de dispersión que filtra por región, un gráfico de pastel que representa la distribución de ventas por región, y una gráfica de barras que muestra el conteo de productos por categoría. Las paletas de colores variadas facilitan la diferenciación visual de las categorías y regiones.

Conclución¶

Este tablero proporciona una vista integral de las ventas y el rendimiento de diferentes categorías y regiones, facilitando la identificación de tendencias y áreas clave de mejora. La interactividad y los filtros permiten una exploración detallada de los datos, haciendo más accesible la toma de decisiones informadas.
